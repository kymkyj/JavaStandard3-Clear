### 정리방법
1. 기술별 폴더로 정리한다.
2. Readme는 각 카테고리에 대한 설명을 간략하게만 정리
3. 티스토리에 공부한 내용을 정리한다.

### 서버세팅
- AWS EC2를 사용하여 별도의 디비를 따로두지않고, EC2 안에 다 세팅하는 방향으로 진행한다
- Tomcat 사용해서 war파일올려서 배포한다.
- 품종에 대한 데이터는 미리 크롤링하여 넣는다.
