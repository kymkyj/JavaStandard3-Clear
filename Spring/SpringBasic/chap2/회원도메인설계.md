### 회원 도메인 설계
* 회원 도메인 요구사항
  * 회원은 가입하고 조회할 수 있다.
  * 회원은 일반과 VIP 두 가지 등급이다.
  * 회원 데이터는 자체 DB를 구축할 수 있고, 외부 시스템과 연동할 수 있다. (미확정)

----

회원 도메인 협력 관계

              회원가입                      |   |   |
              회원조회                  -----   |   -----
                           메모리        DB       외부시스템연동
                               회원저장소    회원저장소     회원저장소
